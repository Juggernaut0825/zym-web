# ZYM

AI-native lifestyle platform. Starting with health.

## About

ZYM is a conversational AI coach that combines nutrition tracking, training support, and social community—all in one place. Built for students and young adults who want healthier, more structured lifestyles.

### Key Features

- **AI Nutrition Coach**: Snap food photos for instant calorie and macro estimates
- **AI Training Coach**: Conversational workout logging with calorie burn tracking
- **Memory-Based Coaching**: Personalized support that remembers your goals and preferences
- **Social Community**: Add friends, share activities, stay accountable together

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
